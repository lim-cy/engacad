Making This Website!

Making this website was like revisting an old friend. The last time I had touched web development was in Sec 2. I had to go reread and relearn a lot of stuff. Downloading Github Desktop made updating the website a lot easier. I didn't need to keep uploading files to Github and it only took a few quick clicks to update the files in Github. As it automatically senses any changes I made, I only need to commit and then push to make the change.


I used the Bootstrap Grid System to have my images and text side-by-side in the projects post. Bootstrap is also used for the navigation bar. Since the Projects page is in a blog layout. I intend to just keep adding new "posts" by copy-pasting and then editing sections of the code for new entries. I used Google Fonts for more font options.

05 April 2021

Engineering Academy 2-Day Experience

During the 2-day experience, we got to print our own designs on T-shirts. I also made a 3D chess piece in Fusion360 - learnt how to use the Revolve command. The Fusion360 lesson was a fun revision as I had previously covered most of the tools in CADD. It was nice to just have fun with the software and not be stressed about doing up a nice assignment. Printing our own designs were really special as we got to customize our shirts - making each of them unique. Designing it together with my tablemates made for a whole load of fun as we got to see each other's ideas and bounce off one another.
